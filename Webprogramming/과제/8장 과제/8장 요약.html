8장 요약

  • HTML 페이지와 JS 객체
    • 자바스크립트 코드는 브라우저로부터 3가지 유형의 객체를 제공받아 활용할 수 있다.
  • HTML DOM
    • 웹 페이지에 작성된 HTML 태그 당 객체 생성
  • DOM 트리
    • DOM 트리는 부모 자식 관계
    • DOM 트리의 루트는 document 객체, 종류는 HTML 태그 종류 만큼, HTML 태그당 DOM 객체가 하나씩 생성됨
  • DOM 객체
    • DOM 트리의 한 노드, HTML 태그 당 하나의 DOM 객체 생성
  • HTML 태그
    • 엘리먼트 이름(태그명), 속성, CSS3 스타일, 이벤트 리스너, 콘텐츠
  • DOM 객체의 구성
    • 프로퍼티, 메소드, 컬렉션, 이벤트 리스너, CSS3 스타일
  • DOM 객체 다루기
    • DOM 객체 구분, id 태그 속성
    • DOM 객체 찾기, document.getElementById()
    • DOM 객체의 CSS3 스타일 동적 변경
  • innerHTML 프로퍼티
    • 시작 태그와 종료 태그 사이에 들어있는 HTML 콘텐츠
    • innerHTML 프로퍼티 수정 > HTML 태그의 콘텐츠 변경
    • ex) let p = document.getElementById("firstP) p.innerHTML = "나의 <img src = 'puppy.jpg'>입니다."
  • this 키워드
    • 객체 자신을 가리키는 JS 키워드
    • DOM 객체에서 객체 자신을 가리키는 용도로 사용
  • document 객체
    • HTML 문서 전체를 대변하는 객체
      • 프로퍼티 - HTML 문서의 전반적인 속성 내포
    • DOM 객체를 접근하는 경로의 시작점, DOM트리의 최상위 객체
    • document 객체는 DOM 객체가 아님
    • getElementsByTagName(), getElementsByClassName()
  • HTML 페이지 로딩 과정
    • 브라우저는 HTML 페이지 로드 전 빈 상태 document 생성
    • HTML 페이지를 위에서 아래로 해석
    • HTML 태그들을 document 객체에 담아 간다(DOM 객체 생성)
    • </html> 태그를 만나면 document 객체를 완성하고 닫는다.
  • document.writeln() 은 한칸 띄는것 뿐, 다음 줄로 넘어가지 않는다.
  • window.open()
    • window는 BOM 객체의 최상위 객체
    • 새 윈도우를 열고 그곳에 HTML 문서 출력
  • 문서의 동적 구성
    • DOM 객체 동적 생성: document.createElement(”태그 이름”);
      • 태그 이름의 DOM 객체 생성
      • ex) let newDIV = document.createElement(”div”);
        newDIV.innerHTML = “새로 생성된 DIV입니다.”;
        newDIV.setAttribute(”id”, “myDiv”); // id 부여
        mewDIV.style.backgroundColor = “yellow”;
      • <div id=myDiv style=”background-color: yellow”>
        새로 생성된 DIV입니다.
        </div>
    • DOM 트리에 삽입
      • 부모.appendChild(DOM 객체); >>>마지막에 추가
      • 부모.prepend(DOM 객체); >>> 앞에 추가
      • 부모.insertBefore(DOM객체, 기준 노드); >>> 앞에 추가
    • DOM 객체의 삭제
      • let removeObj = 부모.removeChild(떼어내고자 하는 자식 객체);
