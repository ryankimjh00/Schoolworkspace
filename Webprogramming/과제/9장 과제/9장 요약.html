9장 요약

  • 이벤트 개요
    • 사용자의 입력 행위나 문서나 브라우저의 상태변화를 자바스크림트 코드에게 알리는 통지
  • 이벤트 리스너
    • 발생한 이벤트에 대처하기 위해 작성된 JS 코드
    • onmouseover, onclick..등등
  • 이벤트 객체
    • 발생한 이벤트에 관련된 다양한 정보를 담은 객체
    • 이벤트가 처리되고 나면 이벤트 객체 소멸
    • 전달받기
      • 이름을 가진 이벤트 리스너
        • function f(e)
      • 익명 함수
        • obj.onclick = function(e){}
      • HTML 태그에 이벤트 리스너
        • function f(e) {} ... <button onclick = "f(event)">...
  • 이벤트 객체에 들어있는 정보
    • 현재 발생한 이벤트에 관한 다양한 정보
      • 이벤트 객체의 프로퍼티와 메소드로 알 수 있음
    • 이벤트 객체의 공통 멤버(type, target, currentTarget, defaultPrevented, reventDefault())
    • target 프로퍼티
      • 이벤트 타겟 객체를 가리킴
  • 이벤트 흐름
    • 이벤트가 발생하면 window 객체에 먼저 도달, DOM트리를 따라 타겟에 도착, 다시 반대로 window 객체에 도달, 소멸
    • 캡쳐 단계
    • 버블 단계
  • DOM 객체의 이벤트 리스너
    • 캡쳐 리스너와 버블 리스너를 모두 소유 가능
  • 캡쳐 리스너와 버블 리스너 등록
    • addEventListener()의 3번째 매개변수 이용
      • true이면 캡쳐, false면 버블
  • 마우스 핸들링
    • onmousedown, onmouseclick, onmousemove 등등
  • onload로 공고창 출력
  • 이미지 로딩 완료와 onload
    • Image 객체
      • < img > 태그에 의해 생성되는 DOM 객체
      • new Image(); JS에 의해 생성되는 객체
  • onfocus와 onblur
    • 입력 없이 다른 창 이동 불가
  • 라디오 버튼과 체크 박스
    • 라디오 버튼은 한개만, 체크 박스는 여러개 선택 가능
  • 키 이벤트
    • onkeydown(키가 눌리는 순간), onkeypress(문자키와 반드시 조합), onkeyup(키를 떼는 순간)
