10장, 11장 요약

  • BOM 객체들
    • 자바스크립트로 브라우저를 제어하기 위해 지원되는 객체들
    • 브라우저 공통 BOM 객체들과 기능
      • window, navigator, history, location, screen
    • window 객체
      • 열려있는 브라우저 윈도우나 탭 윈도우의 속성을 나타내는 객체
      • 브라우저 윈도우나 탭 윈도우마다 별도의 window 객체 생성
      • 자바스크립트 코드로 윈도우 객체에 대한 접근: window
    • window 객체의 타이머 기능: setTimeout("Code/Function", msec)/clearTimeout(): 1회 호출, setInterval()/clearInterval(): 반복 호출. 예제: 텍스트 회전
    • 윈도우 위치 및 크기 조절: window.moveBy(px,px), 이동: window.moveTo(x,y), 재조정: window.resizeBy(px,px), 재조절: window.resizeTo(px,px)
    • navigator 객체
      • 현재 작동중인 브라우저에 대한 다양한 정보를 나타내는 객체
    • screen 객체
      • 브라우저가 실행되는 스크린 장치에 관한 정보를 담고 있는 객체
    • history 객체
      • 윈도우에서 방문한 웹페이지 리스트를 나타내는 객체
  • Canvas 그래픽
    • 웹페이지에 그래픽을 출력하는 <img>나 플러그인을 이용하는것과는 다른 기능
    • 웹페이지에 캔버스 영역 만들기: <canvas id="id" style="style" width="width" height="height">
    • 도형 그리기
      • 과정
        • 경로(path) 만들기 > 캔버스에 경로에 담긴 도형 모두 그리기
        • context.beginPath(); 빈 경로 구성> context.moveTo(120, 20); 시작점 설정> context.lineTo(20,50); context.lineTo(150,120); 경로에 도형 추가> context.stroke(); 캔버스에 그리기
        • context.strokeStyle='blue' 색상 지정
      • 원호 그리기(arc() 메소드)
        • arc(x, y, radius, startAngle, endAngle, anticlockwise) anticlockwise는 시계/반시계, endAngle : 3시 기준에서 그리기 시작함. Math.PI
      • 사각형 그리는 컨텍스트 메소드
      • 캔버스에 그려진 그래픽을 모두 지울때
        • context.clearRect(0,0, canvas.width, canvas.height); 경로는 못지움
        • context.clearRect(0,0, canvas.width, canvas.height); context.beginPath() //경로까지 모두 지움
      • 칠하기
        • fillStyle 프로퍼티: 원호나 사각형, 텍스트의 내부를 칠할 색 지정
        • fillRect(): 캔버스에 바로 칠하기
        • 경로에 닫힌 도형 칠하기: fill()
      • 캔버스에 텍스트 그리는 두가지 방법
        • strokeText(text, x, y [, maxWidth]) 외곽선만 그림
        • fillText(text, x, y [, maxWidth]) 외곽선 없이 다 칠함
      • 이미기 객체 생성
        • img.onload = function(){ context.drawImage(); ... }
